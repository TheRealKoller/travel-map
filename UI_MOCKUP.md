## Key Visual Elements

### Transit Step Card
- **Colored Badge**: Shows the transit line color (e.g., orange for S5)
- **Line Information**: Displays line number/name (S5) and headsign (Westkreuz)
- **Departure Stop**: Green pin icon with stop name and departure time
- **Journey Info**: Arrow icon showing number of stops and duration
- **Arrival Stop**: Red pin icon with stop name and arrival time

### Walking Steps
- **Walking Icon**: Person walking icon
- **Duration and Distance**: Simple text showing time and distance
- **Muted Styling**: Gray background to differentiate from transit

### Alternative Routes
- **Compact List**: Shows other routing options
- **Key Metrics**: Distance, duration, and number of transfers
- **Easy Comparison**: Allows users to see if there are better options

## Color Scheme
- **Transit Line Colors**: Use actual line colors from Google Maps API
- **Green Pin**: Departure stops
- **Red Pin**: Arrival stops
- **Blue Background**: Transit details section
- **Gray Background**: Walking steps
- **Muted Background**: Alternative routes section
