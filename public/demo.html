AI-Enriched Marker Feature Demo

Feature: Markers that have been enriched using AI (Le Chat) are now marked with a purple lightning bolt icon ⚡ to indicate they contain AI-generated information.

1. Marker Form (Edit Dialog)

When viewing an AI-enriched marker in the edit dialog, you'll see an "AI enriched" badge next to the title:

Marker Details AI enriched
This badge appears when a marker has been enriched using the "Enrich with AI" button, which automatically fills in type, UNESCO status, notes, and URL information.

2. Marker List

In the marker list on the left side, AI-enriched markers show a purple lightning bolt icon:

  • Eiffel Tower
    Lat: 48.858400, Lng: 2.294500
  • Local Restaurant
    Lat: 48.860100, Lng: 2.337600
  • Louvre Museum
    Lat: 48.860600, Lng: 2.337600

Notice: Markers with AI-enriched data (Eiffel Tower, Louvre Museum) have a purple lightning bolt ⚡ icon, while regular markers (Local Restaurant) do not.

3. Implementation Details

  • Database: New ai_enriched boolean column added to markers table (defaults to false)
  • Backend: Flag is automatically set to true when user clicks "Enrich with AI" button
  • Frontend: Purple lightning bolt icon displays in both marker list and edit form
  • Tests: 7 new tests verify the functionality works correctly
